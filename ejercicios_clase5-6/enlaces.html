HTML - ENLACES

En HTML, un enlace es una etiqueta que hace que el elemento al
que rodea sirva para acceder a otra instancia interna o externa de
nuestra aplicación web. Para ello usamos <a>, que proviene de la
palabra "anchor" (ancla)

Ejemplos de enlaces

  • Enlace en una pestaña nueva

    Esto se consigue mediante el atributo target="_blank"

  • Enlace la pestaña actual

    Esto se consigue mediante el atributo target="_self"

  • Enlace mediante la etiqueta <iframe>:

    Muestra el contenido dentro de un marco en la propia
    página, se utiliza el atributo src="" para definir qué página
    enlazar.

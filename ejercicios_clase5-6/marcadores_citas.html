Marcadores de citas en HTML

HTML - MARCADORES DE CITAS


Sirven para resaltar una parte del texto mediante el renderizado,
de forma que el texto destaque con letra negrita o la letra cursiva,
aludiendo así a una fuente externa.

  • El uso de la etiqueta <blockquote>:

    El uso de la etiqueta <blockquote> sirve para cuando
    queremos citar un texto perteneciente a un sitio externo,
    del cual se indica la fuente, por ejemplo:

    Afectas al mundo por lo que navegas
    Tim Berners-Lee

  • El uso de la etiqueta <cite>:


    El uso de la etiqueta <cite> es una etiqueta de línea que se
    usa tanto como también como atributo de <blockquote> y
    renderiza su contenido en negrita. Se usa para referenciar
    un trabajo creativo:

    Tejiendo la red

    Tim Berners-Lee

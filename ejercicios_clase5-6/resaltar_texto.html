HTML - RESALTAR TEXTO


Son resaltadores de texto para darle mayor énfasis o destacar a las
partes del texto que nos interesen mediante estas tres etiquetas.

  • El uso de la etiqueta <code>:

    El uso de la etiqueta <blockquote> sirve para cuando
    queremos citar un texto perteneciente a un sitio externo, del cual
    se indica la fuente, por ejemplo:

    Afectas al mundo por lo que navegas
    Tim Berners-Lee

  • El uso de la etiqueta <strong>:


    El uso de la etiqueta <cite> es una etiqueta
    de línea que se usa tanto como también como atributo de <blockquote>
    y renderiza su contenido en negrita. Se usa para referenciar un trabajo
    creativo:

    Tejiendo la red

    Tim Berners-Lee
